# Theme System

The design system supports **two portfolio identities** and **two visual modes**, allowing the interface to adapt its aesthetic depending on the showcased profile.

---

## Portfolio Variants

<div className="section">

<div className="sectionContent">

### Dev Portfolio

Designed for technical and structured content.

**Characteristics:**

- Functional visual language
- Strong readability
- Neutral / productivity-oriented palette
- Optimized for showcasing technical projects and engineering work

## Theme Modes

<div className="section">

<div className="sectionContent">

### Light Mode

Best suited for:

- Bright environments
- Editorial / reading-heavy interfaces
- Softer visual presentation

</div>

<div className="sectionContent">

### Dark Mode

Best suited for:

- Low-light environments
- High contrast presentation
- More immersive / premium aesthetics
