# Icons list

The icons used in this design system come from Lucide.
They are vector-based, lightweight, and designed to integrate consistently within the interface.

They inherit their color via currentColor, allowing seamless integration with color tokens and themes.

## Usage

Icons are used to:

support actions (buttons, navigation)
improve visual understanding
indicate states or categories

They should remain simple and legible, without overloading the interface.

## Size

The default icon size is 16px, which corresponds to the Lucide standard.
The design system defines the following sizes:

xs: 4,
sm: 8,
md: 16,
lg: 24,
xl: 32,

## Exemple

This example shows a basic usage of a Lucide icon.
The icon is imported directly and rendered with a size of 16px, which is the default size in the design system.

import { Home } from "lucide-react";

<Home />
